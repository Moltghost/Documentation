# Core Concepts

MoltGhost is built on several foundational concepts that you need to understand.

## Main Components

1. **Agent Pod** - The container for your agents
2. **Agent Runtime** - The execution environment
3. **Local LLM** - On-premise language models
4. **Agent Access** - Permission and access control
