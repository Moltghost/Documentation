# MoltGhost Documentation

Learn how to deploy, manage, and operate private AI agents with MoltGhost.

MoltGhost provides infrastructure to run fully private AI agents with dedicated compute, local models, and isolated environments.

---

## What is MoltGhost

MoltGhost is a private AI agent infrastructure that gives every AI agent its own isolated environment.

Instead of relying on shared AI services or external platforms, MoltGhost runs agents on dedicated machines where models, tools, and data remain fully private.

Each agent operates independently with its own resources, allowing users to run AI agents securely while keeping their data and activity completely under their control.
